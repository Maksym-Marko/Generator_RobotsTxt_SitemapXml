С помощью данной программы, можно на своем сайте создавать файлы необходимые для индексации страниц, а именно: robots.txt и sitemap.xml.

Инструкция по применению:
1 - папку "generator" закачать в корень сайта;
2 - по адресу http://example.com/generator можно создать файлы robots.txt и sitemap.xml с параметрами, которые Вам необходимы (имя домена, запрет на индексацию директорий администратора, адреса страниц сайта, которые необходимо индексировать…).
